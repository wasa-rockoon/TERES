# Tiny & Extendable Rocket Electronics Stack

モデルロケットに搭載可能な，小型かつ拡張可能な電装


# 要求

- 内径40mmのボディに収まること
- 水密を確保できること
- ミッションや開発状況に応じて組み替え・拡張可能であること


# 仕様

- モジュール式
- マイコン：RP2040

## 外形

- 基板外径：36mm


## バス

上下の基板同士はUARTで接続し，デイジーチェーンで全体にわたるバスを形成する．

1. V+
2. GND
3. RX
4. TX

## モジュール

- LiPo電源・熱管理
- GPS
- IMU（9軸センサ，気圧センサ）
- 高速テレメトリ（TWELITE）
- 長距離テレメトリ（LoRa）
- ログ（SDカード）
- カメラ
